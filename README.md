## Run Locally

Clone the project

```bash
  git clone https://github.com/rushirajput18/AsyncCoders_UniConnect.git
```

Go to the project directory

```bash
  cd agnethon_2024
```
```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React, Bootstrap

**Server:** Node, Express

**Database:** MongoDB
